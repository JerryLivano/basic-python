Apa Itu Python?

Python adalah salah satu bahasa pemrograman yang dapat melakukan eksekusi sejumlah instruksi multi guna secara langsung (interpretatif) dengan metode orientasi objek (Object Oriented Programming) serta menggunakan semantik dinamis untuk memberikan tingkat keterbacaan syntax. Python juga diartikan sebagai bahasa yang kemampuan, menggabungkan kapabilitas, dan sintaksis kode yang sangat jelas, dan juga dilengkapi dengan fungsionalitas pustaka standar yang besar serta komprehensif. Walaupun python tergolong sebagai bahasa pemrograman tingkat tinggi, nyatanya python dirancang sedemikian rupa agar mudah dipahami.

Mengapa Menggunakan Python?

Gratis dan Open Source

Semua library, modul, maupun tools untuk menjalankan Python dapat Anda nikmati tanpa perlu membayar sepeserpun. Selain itu, Python bebas digunakan baik untuk tujuan pribadi maupun komersial, tanpa harus membayar ke pengembangnya.

Mudah Dipelajari

Sebagai bahasa tingkat tinggi dengan sintaks mirip bahasa manusia, Python mudah dipelajari jika dibandingkan bahasa lain. Selain itu, sebagai salah satu bahasa terpopuler, Python punya komunitas yang luas dan suportif.

Multiplatform

Anda bisa leluasa menulis coding Python, lalu menjalankannya di berbagai sistem operasi populer (Windows, Linux, atau MacOS). Anda bisa segera mulai belajar Python, tanpa perlu mengganti atau membeli perangkat baru yang kompatibel dengan Python.

Multifungsi

Walaupun sintaksnya yang tergolong singkat dan mudah, Python juga memiliki banyak fungsi, seperti: membangun aplikasi berbasis desktop, mobile, ataupun web; mengolah data; menganalisis data; dan membangun Machine Learning.

Tipe Data Pada Python

Integer

Integer digunakan untuk memasukkan nilai bilangan bulat. Contohnya: 10, -20, 100, dst. Tipe data ini biasanya digunakan sebagai parameter operasi aritmatika dalam pemrograman.

Float

Sama halnya seperti integer, namun float dituliskan sebagai bilangan desimal. Contohnya: 10.0, -30.4, 0.5, dst.

String

String adalah tipe data Python untuk menampung nilai teks, seperti huruf, tanda baca, dan karakter spesial lainnya. Ciri-ciri tipe data string adalah diapit tanda petik, baik itu petik satu (' ') atau petik dua (" ").

Boolean

Boolean adalah tipe data Python yang hanya bisa diisi oleh dua nilai, yaitu True dan False. Aturan penulisannya adalah dengan menggunakan huruf kapital di huruf pertama dan tanpa menggunakan petik.
